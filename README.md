# 🕹️ Lost Cave

Um jogo 2D desenvolvido em **Python** para a disciplina de Computação Gráfica, inspirado em clássicos como Diamond Rush e Enter the Dungeon, além da estética retrô de jogos de Game Boy.

No jogo, o jogador controla um explorador que precisa atravessar uma caverna repleta de puzzles, obstáculos e desafios para encontrar o tesouro escondido em suas profundezas.

---

# ✨ Funcionalidades

## 🖊️ Rasterização
- Rasterização de linhas
- Rasterização de circunferências
- Rasterização de elipses
## 🎨 Preenchimento
- Flood Fill / Boundary Fill
- Preenchimento Scanline
- Polígonos com gradientes de cor
- Mapeamento de textura

## 🔄 Transformações Geométricas
- Translação
- Rotação
- Escala

## 🎥 Recursos Gráficos
- Sistema de Window e Viewport
- Zoom e movimentação da câmera
- Animações 2D
- Clipping utilizando Cohen-Sutherland

## 🎮 Interação
- Controle por teclado
- Menus interativos
- Área demonstrativa para o professor com exemplos de:
  - Rotação
  - Escala
  - Translação
  - Clipping
  - Viewport

---

# 🗺️ Estrutura do Jogo

O jogo é composto por:

- 🪨 Duas fases principais com puzzles
- 🕷️ Um boss final
- 🏆 Tela de vitória e encerramento
- 🔬 Área técnica para demonstração dos algoritmos gráficos
